# Angular authentication with Azure static web apps

Does not run in webcontainer

## Running locally

To run the static web app locally:

1. Install the Azure Static Web Apps CLI globally:

   ```bash
   npm install -g @azure/static-web-apps-cli
   ```

2. Start the development server:

   ```bash
   swa start
   ```

3. Open your browser and navigate to the URL displayed in the terminal (typically `http://localhost:4280`)
